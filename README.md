# HR Analytics Dashboard

An interactive Tableau dashboard built to analyze employee attrition, job satisfaction, and workforce trends through clear and insightful visualizations.

## Dashboard Highlights
- Tracked HR metrics such as **attrition, employee count, and job satisfaction**
- Analyzed **department-wise, role-wise, and gender-based** workforce trends
- Used interactive charts and filters for easy exploration of HR data
- Generated insights to support workforce analysis and decision-making

## Tools Used
- Tableau
- Data Visualization
- Dashboarding
- HR Analytics

## Key Insights
- Identified attrition patterns across different employee groups
- Compared workforce distribution by department, role, and gender
- Created an interactive dashboard for HR reporting and analysis

## Outcome
Converted raw HR data into a visual dashboard to uncover workforce trends and support data-driven HR decisions.
